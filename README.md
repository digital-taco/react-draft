# react-draft
📝 WIP: Develop your React components in isolation without any configuration

# Installation

In your project's root directory, run:

```bash
npm i @js-draft/react-draft
```

In your project's `package.json`, add:

```json
"scripts": {
  "draft": "npm run draft"
}
```
